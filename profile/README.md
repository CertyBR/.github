# Certy

O **Certy** é um projeto para simplificar a emissão de certificados TLS/SSL e tornar a adoção de HTTPS mais acessível para times e pessoas desenvolvedoras.

Nosso foco é transformar um processo sensível e cheio de etapas em uma experiência clara, segura e confiável.

## Por que essas tecnologias

- **SvelteKit + TypeScript (Frontend):** entrega uma experiência rápida e moderna para quem está emitindo e acompanhando certificados.
- **Cloudflare Workers (Proxy):** adiciona uma camada de proteção e controle entre cliente e backend com baixa latência global.
- **Rust + Axum (Backend):** garante alta confiabilidade e desempenho para o fluxo crítico de emissão.
- **PostgreSQL (Dados):** persistência sólida para sessões, auditoria e estado operacional do sistema.
